# hiit-workout-planner

A workout app. specifically a HIIT (high-intensity interval training) workout app. The app should let you specify the amount of time that you want to workout and then it should also let you indicate if you have any equipment. Based on the the amount of time and what equipment you have it should spit out a workout routine.
